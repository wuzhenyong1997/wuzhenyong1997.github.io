如何在VMware Workstation安装RedHat Linux 7.4

Aug 13, 2018 | 技术文章 | 432 字 | 2 分钟
条评论

这篇文章将要介绍如何在VMware Workstation上安装RedHat Linux 7.4版本。
只要有点耐心，按照如下步骤一步步安装就可以安装成功。

1、创建新的虚拟机

2、选择“自定义（高级）”

3、选择“下一步”

4、选择“稍后安装操作系统”

5、选择操作系统的类型

6、设置虚拟机名称和保存路径

7、选择“下一步”

8、选择“下一步”

9、设置网络类型，选择“使用仅主机模式”

10、选择“下一步”

11、选择“下一步”

12、选择“下一步”

13、设置硬盘容量，如果硬盘空间足够，可以适当大一些

14、选择“下一步”

15、点击“完成”

16、加载ISO光盘
如果没有下载ISO镜像文件，点击如下链接进行下载，下载版本为 rhel-server-7.4-x86_64-dvd.iso
rhel-server-7.4-x86_64-dvd.iso下载地址

17、启动虚拟机

18、选择第一个选项，并回车

19、选择“Continue”

20、选择“设置时区” —> Asia/Shanghai

21、配置安装选项

（1）选择“Server with GUI”

（2）选择硬盘

（3）禁用KDUMP

22、配置网络和主机名



注意：配置IP地址，下处设置的IP地址为生效主机名后生成的IP地址

23、开始安装

24、设置root用户的密码，并等待安装，需要等待一段时间

25、安装完成后，重启虚拟机

26、完成配置

27、完成向导








点击中间的“Start using Red Hat Enterprise Linux Server”按钮就启动了虚拟机。到这里就大功告成啦，如果安装过程中遇到什么问题，欢迎评论。
