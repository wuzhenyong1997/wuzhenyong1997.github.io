Map(映射)是一种可迭代的键值对（key/value）结构，使用操作符”->”来创建。
Map中的键都是唯一的，所有的值都可以通过键来获取。
Map有两种类型，可变与不可变，区别在于可变集合的元素可以修改，不可变集合不可以修改。
默认情况下 Scala 使用不可变 Map。
如果需要使用可变集合，要显式的引入 import scala.collection.mutable.Map 类
